Swap- P2 Gold Showcase

I built this project for my EECS 494 game design class. I completed the project over 2 weeks. I was the sole developer on this project. I developed the "swap" mechanic, where players are able to switch back and forth between platforms.

This is a basic puzzle platformer, with a few enemies that will cause a respawn on touch. The goal is to make it to the gem at the end of each level. Both C# and Unity were used in this project
